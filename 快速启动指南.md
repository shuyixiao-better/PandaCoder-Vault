## 🎯 三步启动后端服务

### 第一步: 配置环境变量

```bash
cd backend
cp .env.example .env
```

然后编辑 `backend/.env` 文件,修改你的数据库配置:

```properties
MONGODB_HOST=你的MongoDB服务器地址
MONGODB_PORT=27017
MONGODB_DATABASE=PandaCoder
MONGODB_USERNAME=你的MongoDB用户名
MONGODB_PASSWORD=你的MongoDB密码
MONGODB_AUTH_DATABASE=admin

JWT_SECRET=你的JWT密钥
JWT_EXPIRATION=86400000

SERVER_PORT=8080
```

**注意**: 密码中的特殊字符(如 `@`)直接写即可,不需要编码!

### 第二步: 启动后端

在项目根目录执行:

```bash
./start-backend.sh
```

启动脚本会自动:
- ✅ 使用 JDK 21
- ✅ 查找可用的 Maven
- ✅ 编译项目
- ✅ 启动服务

## 🐛 遇到问题?

### Maven 找不到?
启动脚本会自动使用 IntelliJ IDEA 自带的 Maven,无需单独安装。

### MongoDB 连接失败?
检查 `.env` 文件中的配置是否正确,特别是密码。

### 端口被占用?
修改 `.env` 文件中的 `SERVER_PORT` 配置。
